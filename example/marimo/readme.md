# Use marimo

## Use marimo to run the demo

## Install requirements
```bash
pip install -r requirements.txt
```

## Test environment
```bash
python 00_env_test.py
```

## Compare models
Run compare model
```bash
marimo run 02_compare_finetune_model.py
```

or
Edit marimo file
```bash
python edit 02_compare_finetune_model.py
```
